# Book MS
![Book MS](static/logo.png)

## 動作環境
- Web: Nuxt.js
- Store: Vuex 
- Style: Element UI
- Hosting: Firebase Hosting
- DB: Cloud Firestore
- VCS: Git + GitHub
- CI/CD: GitHub Actions

## ビルドとセットアップ

``` bash
# パッケージのインストール
$ yarn install

# 開発環境で実行 (localhost:3000)
$ yarn dev

# ビルドして実行
$ yarn build
$ yarn start

# アプリケーションをビルドして静的ファイルを生成
$ yarn generate
```
